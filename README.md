# Dot Space Game

#### Dot Space game in javascript.

 1. Avoid RED dots.
 2. Touch GREEN dots for invulnerability.
 3. Use invulnerability to destroy RED dots.
 4. Score EXTRA points by moving around a lot.
 5. STAY ALIVE !
